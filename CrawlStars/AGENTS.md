# Project guidance

## API reference contracts

- REST API 또는 HTTP 네트워크 코드를 구현하거나 검토하기 전에 `Docs/References/API/openapi.yaml`을 확인해요.
- WebSocket 연결, 메시지, 이벤트 또는 실시간 동기화 코드를 구현하거나 검토하기 전에 `Docs/References/API/asyncapi.yaml`을 확인해요.
- 요청 및 응답 필드명, 대소문자, 타입, 상태값과 메시지 흐름은 해당 명세를 기준으로 맞춰요.
- 코드와 명세가 충돌하면 임의로 한쪽을 선택하지 말고 충돌 내용을 먼저 알려요.
